# Personal Website

A simple, modern, responsive personal website built with HTML, CSS, and JavaScript.

## Files

- `index.html` - Page structure and content
- `styles.css` - Responsive styling and layout
- `script.js` - Mobile navigation and dynamic footer year

## Run Locally

Open `index.html` directly in your browser.

You can also run a local static server from this folder:

```bash
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## Customize

- Replace `A. Velaga` with your full name.
- Update the About, Work, and Contact sections.
- Change `hello@example.com` to your real email address.
- Swap the hero image URL in `styles.css` if you want a different visual style.
