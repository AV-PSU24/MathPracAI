# Prompts

## Purpose

The Prompts folder contains all editable prompt assets that define Milo's behavior.

## Responsibilities

- System prompts
- Runtime prompts
- Action prompts
- Prompt schemas
- Prompt examples

## Philosophy

Prompt behavior belongs in text files, not Python code. Python assembles prompts and selects which prompts to use, while prompt files define how Milo behaves. This separation allows prompt engineering without modifying application logic.
