# Math Engine

## Purpose

The Math Engine is the mathematical core of MathPracAI and the source of truth for every generated problem.

## Responsibilities

- Generate problems
- Render mathematical content
- Validate answers
- Format mathematical output
- Enforce problem contracts

## Philosophy

The Math Engine determines mathematical truth. Nothing outside this package should modify mathematical answers or force generated problems to satisfy selected options. Every problem must be mathematically correct before it reaches the student.
